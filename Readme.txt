# Sierra | Michael Ibangha Portfolio

This repository contains the premium digital portfolio of **Michael Ibangha**, Chief Technology Officer at Africentric Technology Hub.

## Project Overview
The portfolio has been redesigned to feature a high-performance, modern aesthetic with a focus on:
- **Premium Design**: Dark mode aesthetic with vibrant red accents and glassmorphism.
- **Modern Stack**: Built with HTML, Vanilla CSS, and Tailwind CSS.
- **Responsive Layout**: Optimized for all devices, from desktop to mobile.
- **Interactive Sections**: Includes Hero, About, Services, Portfolio, Testimonials, and Contact sections.

## Key Features
- **Single-Page Experience**: Seamless navigation through all sections.
- **Professional Identity**: Integration of professional bio, tech stack, and experience.
- **High Impact**: Visuals engineered for engagement and conversion.
